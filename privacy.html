Zipline Android Privacy Policy

This privacy policy explains how Zipline Android collects, uses, and protects your information when you use our app.

Access

In order to use the Zipline Android app, the user must have a self-hosted Zipline instance accessible over HTTP(s) and have the login credentials for the aforementioned instance.

Use

The app allows its users to upload and access the files and shortened URLs on their self-hosted Zipline instance. All interactions occur directly between the user's device and the Zipline instance configured during login. The Zipline Android app does not collect any data from user interactions.

Login Data

Upon login, the Zipline Android app stores the user's access token on the user's device in the app's private storage. This token is used solely to keep the user logged in and avoid repeated authentication. This token is only shared with the user's Zipline instance to authenticate the HTTP requests.

Ads

The Zipline Android application does not contain any avertisement. The app only displays the data returned by the user's Zipline instance.

Contact

If you have any question about this privacy policy, please contact the following email address: privacy@zipline.stefdp.com.
